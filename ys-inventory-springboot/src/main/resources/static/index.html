Google Sheets Setup Guide

~5 min

1 Open Google Apps Script

Go to script.google.com → Click New Project → Delete any existing code.

2 Paste this Code

⚠️ Already set up before? You MUST do this:
1. Go to script.google.com → Open your project
2. Delete ALL old code completely
3. Paste the NEW code below (it now supports JSONP — fixes mobile)
4. Replace YOUR_SHEET_ID_HERE with your Sheet ID
5. Click Deploy → New Deployment (must be NEW, not edit old)
6. Copy the new URL and paste it here
// ✅ FINAL SCRIPT — Copy this EXACTLY. Works on Mobile + Desktop. // Key: respond() supports JSONP callback for mobile browsers. function doGet(e) { var cb = e.parameter.callback || ""; try { var raw = e.parameter.data; if (!raw) return respond(cb, {status:"error", message:"No data"}); var d = JSON.parse(decodeURIComponent(raw)); var ss = SpreadsheetApp.openById(SHEET_ID); if (d.action === "ping") return respond(cb, {status:"ok", message:"Connected successfully!"}); if (d.action === "saveAll") { save(ss, "Products", d.products || []); save(ss, "Bills", d.bills || []); save(ss, "Customers", d.customers || []); save(ss, "Settings", [d.settings]); return respond(cb, {status:"ok"}); } if (d.action === "loadAll") return respond(cb, { status: "ok", products: load(ss, "Products"), bills: load(ss, "Bills"), customers: load(ss, "Customers"), settings: load(ss, "Settings") }); return respond(cb, {status:"error", message:"Unknown action: "+d.action}); } catch(err) { return respond(cb, {status:"error", message:err.toString()}); } } // respond() — if callback name given, wraps in JSONP; else returns plain JSON function respond(cb, obj) { var body = JSON.stringify(obj); if (cb) body = cb + "(" + body + ")"; return ContentService .createTextOutput(body) .setMimeType(ContentService.MimeType.JAVASCRIPT); } function save(ss, name, arr) { var sheet = ss.getSheetByName(name) || ss.insertSheet(name); sheet.clearContents(); if (arr && arr.length > 0) sheet.getRange(1, 1, arr.length, 1) .setValues(arr.map(function(o){ return [JSON.stringify(o)]; })); } function load(ss, name) { var sheet = ss.getSheetByName(name); if (!sheet || sheet.getLastRow() === 0) return []; return sheet.getRange(1, 1, sheet.getLastRow(), 1) .getValues() .map(function(r){ try { return JSON.parse(r[0]); } catch(e) { return null; } }) .filter(Boolean); } // 👇 REPLACE THIS with your Google Sheet ID (the long code from the URL) var SHEET_ID = "YOUR_SHEET_ID_HERE";

3 Create Google Sheet & Get ID

  1. Go to sheets.google.com → New blank sheet
  2. Name it YS Chemical Agencies Data
  3. Copy the ID from the URL:
    ...spreadsheets/d/THIS_PART/edit
  4. Replace YOUR_SHEET_ID_HERE in the code

4 Deploy as Web App

  1. In Apps Script, click Deploy → New Deployment
  2. Click ⚙️ gear icon → Select type: Web App
  3. Description: type anything e.g. YS Chemical v2
  4. Execute as: Me (your Gmail)
  5. Who has access: Anyone
  6. Click Deploy → Click Authorize access → Choose your Google account → Click Allow
  7. Copy the Web App URL shown (starts with https://script.google.com/macros/s/...)
Already deployed before? You must create a New Deployment (not edit existing). Old URL will not work with new code.

5 Paste URL & Test

  1. Paste the new Web App URL in the field on the left
  2. Tap Save & Connect
  3. Tap Test Connection — you should see ✅ Connected!
  4. If it still fails: go back to Apps Script → Deploy → Manage Deployments → confirm it shows Anyone in access
